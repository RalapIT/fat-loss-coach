## 使用说明

- 标注 `熟` 的为熟重数据（如米饭、面条），其余为生重
- 外食估算条目已含油盐调味，直接使用
- 加工食品以常见品牌均值为准，建议以包装标签为准
- 营养素缩写：**热量(kcal)｜碳水(g)｜蛋白质(g)｜脂肪(g)**

## 九、查询决策逻辑

```
用户报告食物
    │
    ▼
在此表查询？
    │
   是 ──→ 直接使用表中数据计算
    │
   否 ──→ web_search 查询"[食物名] 每100g营养成分"
              │
              ▼
          找到权威数据？（中国食物成分表 / 官方营养标签）
              │
             是 ──→ 使用数据 + 在备注注明"来源：web_search"
              │
             否 ──→ 使用相似食物估算 + 标注"⚠️ 估算值，误差较大"
```
