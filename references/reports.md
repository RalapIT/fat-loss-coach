# 日报 / 周报 / 月报生成逻辑

所有数据来源必须从飞书表格读取，禁止使用聊天上下文。

---

## 日报（每天 21:00 触发）

### 数据查询

1. 查**每日计划表**当天记录：碳日类型、各营养素目标、实际摄入、差值
2. 查**训练记录表**当天记录：训练类型、时长、消耗热量
3. 查**每日计划表**明天记录：碳日类型、营养素目标

### 报告格式

```
📊 [日期] 日报

🏋️ 训练
  类型：[训练类型 or 休息]
  时长：[X分钟] | 消耗：[Xkcal]

🍽️ 饮食（[碳日类型]）
  热量：[实际] / [目标] kcal  差值：[+/-X]
  碳水：[实际] / [目标] g     差值：[+/-X]
  蛋白质：[实际] / [目标] g   差值：[+/-X]
  脂肪：[实际] / [目标] g     差值：[+/-X]

📈 今日评分
  [根据差值给出：✅达标 / ⚠️轻微超标 / ❌超标 / 📉摄入不足]

💪 进阶建议
  [查动作明细表，找出近3次训练中同一动作都完成了目标组数×次数的，提示可以加重]
  示例："卧推 70kg 已连续完成3次，下次可以尝试 72.5kg"
  无进阶信号时：不显示此栏

💡 明日建议
  明天是[碳日类型]，目标热量[X]kcal
  [1-2句具体建议]
```

### 进阶建议查询逻辑

查**动作明细表**，按动作名称分组，找出：
- 同一动作在最近 3 条记录中重量相同
- 且每次组数 × 次数都达到或超过上次

满足条件 → 建议加重 2.5~5kg（大肌群加 5kg，小肌群加 2.5kg）
不满足 → 不显示进阶建议

### 评分规则

| 热量差值 | 评分 |
|---------|------|
| -100 ~ +200 kcal | ✅ 达标 |
| +200 ~ +400 kcal | ⚠️ 轻微超标 |
| > +400 kcal | ❌ 超标 |
| < -300 kcal | 📉 摄入不足（可能影响训练恢复） |

## 周报（每周日 21:00 触发）

### 数据查询

1. 查**每日计划表**本周（周一到周日）所有记录：汇总实际摄入
2. 查**训练记录表**本周所有记录：统计训练天数、总消耗
3. 查**体重体脂记录表**本周最新一条（如有）

### 报告格式

```
📊 [日期范围] 周报

🏋️ 训练完成率
  完成：[X]天 / 计划：[X]天（[完成率%]）
  本周总消耗：[X]kcal

🍽️ 饮食执行
  平均每日热量：[X]kcal（目标均值：[X]kcal）
  达标天数：[X]天 / 7天
  最难控制的营养素：[碳水/脂肪/蛋白质]

⚖️ 体重变化
  [本周体重 or "本周未测量，建议明天早晨空腹测量"]

📈 进度
  距目标体脂还需减脂：[X]kg（按当前速度约[X]周）

💡 下周建议
  [1-3条具体调整建议]
```

## 月报（每月最后一天 21:00 触发）

### 数据查询

1. 查**体重体脂记录表**本月所有记录：体重/体脂趋势
2. 查**每日计划表**本月所有记录：饮食执行率
3. 查**训练记录表**本月所有记录：训练量统计
4. 查**动作明细表**对比月初/月末重量（力量进步）

### 报告格式

```
📊 [月份] 月报

⚖️ 体成分变化
  体重：[起始] → [当前] kg（[+/-X]kg）
  体脂率：[起始] → [当前] %（[+/-X]%）
  净肌量：[起始] → [当前] kg

🏋️ 训练统计
  训练天数：[X]天
  总消耗：[X]kcal
  力量进步：[举例：卧推 70kg → 75kg]

🍽️ 饮食执行率
  达标天数：[X] / [X]天（[%]）
  平均每日热量：[X]kcal

📈 减脂进度
  本月减脂：[X]kg 纯脂肪
  距目标还差：[X]kg（预计[X]周达标）

💡 下月方案调整
  [根据进度决定：维持方案 / 加大缺口 / 增加训练频率 / 调整碳日分配]
```

---

## 触发时间（写入 HEARTBEAT.md）

- 日报：每天 21:00，检查当天是否有训练/饮食记录，有则生成
- 周报：每周日 21:00
- 月报：每月最后一天 21:00

发送目标：飞书私聊（当前会话）
